Redis-MySQL Consistency Auditor & Visualizer

Project Overview

The Redis-MySQL Consistency Auditor & Visualizer is a Python-based project that checks the consistency between MySQL database records and Redis cache data. It helps identify mismatched records and generates an audit report along with a graphical heatmap for visualization.

Features

- MySQL Database Integration
- CRUD Operations
- Tkinter GUI
- MySQL Data Retrieval
- NumPy-based Data Comparison
- Matplotlib Consistency Heatmap
- JSON Audit Report Generation
- Redis Integration (Code Ready)

Technologies Used

- Python
- MySQL
- Redis
- Tkinter
- NumPy
- Matplotlib
- mysql-connector-python

Installation

1. Clone the repository.
2. Install the required libraries:
   pip install -r requirements.txt
3. Configure MySQL database.
4. Run the Python files.
